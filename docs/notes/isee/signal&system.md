## 课程总结

### 关于课程
*信号与系统中文班*

>分数构成：平时（作业+Matlab+课堂表现）35% + 课程设计35% + 闭卷考试30% 

发现其实课程设计占比比考试还大，而且课程设计的要求比较人性化，ddl在考试之前，也就是期末周有大把的时间可以拿来搓课程设计+写报告:yum:

课程平时的要求比较少，作为一门四学分的专业课相当友好，唯一的槽点就是一次布置一章的作业（腹泻式布置，一次作业能做好久）；期末考也不是很为难人，平均一个小时出头就能做完一份卷子，我直接考试前一天疯狂拟合4份历年卷，感觉颇有成效

### 关于老师

<del>都说隔壁班hhj老师非常好，我偏要来吹一吹我们郑斌老师</del>

老师温文尔雅，讲课时经常笑咪咪的，讲课水平高，有重点、进度适中（据说隔壁班hhj老师讲课虽然细但是进度比较慢）；

我觉得郑老师班非常好的一点是理论课和实验课的分离：理论部分由郑老师上，实验部分由赵老师上；理论部分可以专注于课内的知识（面向考试），而实验部分更强调自己探索琢磨MATLAB，中间有一个小的展示，最后还有一个大的课程设计。赵老师人也非常好，非常鼓励我们自己探索和上台展示，即使展示的效果不是很好也会给我们很高的分数。

两位老师给分都很好，出分之前我一直紧张紧张紧张紧张紧张，因为不是很有底，结果出分直接给我一个大惊喜:angel:
